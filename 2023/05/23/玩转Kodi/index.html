Kodi 简介

Kodi（以前称为XBMC）是一个免费的开放源代码媒体播放器软件应用程序，由XBMC基金会（一家非营利技术联盟）开发。 Kodi可用于多种操作系统和硬件平台，它允许用户播放和查看大多数流媒体，如来自Internet的视频，音乐，播客和视频，以及来自本地和网络存储媒体的所有常见数字媒体文件。

Kodi 安装

打开kodi的下载页面，可以看到它支持多种平台：Windows，Linux，Android，Raspberry Pi，macOS，iOS，tvOS。由于苹果系统的封闭性，导致在苹果的操作系统上安装软件非常复杂，Kodi下载页面提供了非常详细的安装步骤，条件允许的情况下可以照着步骤尝试操作。Windows和Android下，都是常规步骤。

中文设置

默认皮肤切换中文

  1. 在kodi的主页面，找到并且点击页面左上方的设置图标。
  2. 在新的页面找到并点击 Interface。
  3. 在Skin下点击 Fonts，在弹窗中，找到并点击 Arial based，自动关闭窗口。
  4. 在Regional下点击 Languages，在弹窗中选择 Chinese(Simple)。

至此，页面显示为中文。需要注意的是，Kodi的大部分皮肤插件中没有Arial based会导致整个页面乱码，更换皮肤显示中文的方案待完善。

添加视频数据

通过设置一个视频源来使Kodi对其进行扫描，添加到软件中。

添加本地视频

按照如下方式添加：
视频（Videos）->文件（Files）->添加视频（add videos）->浏览（Browse）->[选择存放文件的位置]->设置文件类型[电影/剧集/无]
添加成功后，就可以在kodi中打开并观看视频了，如果剧集和电影分别设置了文件夹，就可以从对应的（电影/剧集）入口观看了。
如果需要在对应的板块展示海报墙，见下方的内容。

添局域网视频

按照如下方式添加：
视频（Videos）->文件（Files）->添加视频（add videos）->浏览（Browse）->添加网络位置（Add network location）->[键入服务器名称（IP地址）、用户名和密码]->[选择刚刚添加的位置]->设置文件类型[电影/剧集/无]
添加成功后，就可以在kodi中打开并观看视频了，如果剧集和电影分别设置了文件夹，就可以从对应的（电影/剧集）入口观看了。
如果需要在对应的板块展示海报墙，见下方的内容。

局域网内多端同步

默认情况下，多台Kodi机器不会彼此识别。在一台电视上观看的剧集不会在另一台电视上显示为“观看”。 我们可以通过MySQL数据库来实现多端的数据同步功能，观看记录等信息都会在多客户端之间同步保存。

安装Mysql

使用局域网内或者公网上电脑（服务器）安装一个MySQL，建议版本为5.7。安装方式比较简单，网上的操作步骤有很多，请自行百度安装。

New-NetFirewallRule -DisplayName "Allow inbound TCP Port 3306 for MySQL" -Direction inbound –LocalPort 3306 -Protocol TCP -Action Allow

备份Kodi

默认情况下，Kodi使用内部SQLite数据库。 为了使Kodi能够在您的家庭网络中有效地进行通信，需要指示它使用外部MySQL数据库。不过在此之前，需要将当前Kodi的配置从SQLite中备份下来。配置路径在专家（expert）模式：
设置（Settings）->媒体（Media）->资料库（Library）->导出资料库（Export library）
可以将库导出为单个文件或多个文件。 单个文件选项是将备份放在一个位置，而多个文件会将多余的JPG和NFO文件分散到媒体文件夹中，这很混乱（不建议）。选择需要保存到的路径并记住这个路径，稍后导入时需要。

Kodi使用MySQL

为了将Kodi指向MySQL，需要编辑Kodi的advancedsettings.xml文件。默认情况下，此文件不存在。但是如果存在advancedsettings.xml文件，则文件将位于以下位置：

  • Windows: C:\Users[username]\AppData\Roaming\Kodi\userdata
  • Linux and other Live versions of Kodi: $HOME/.kodi/userdata
  • macOS: /Users/[username]/Library/Application Support/Kodi/userdata
  • Android: /data/org.xbmc.kodi/files/.kodi/userdata/ (In some Android setups the path may be slightly different to the one stated above.)
  • tvOS: /private/var/mobile/Library/Preferences/Kodi/userdata/

注意事项

如果Mysql安装在SMB的服务器上，那么Kodi添加视频的时候一定记得不要使用本地视频作为源，和其他平台一样，设置SMB链接，通过SMB链接添加视频源。否则会导致其他客户端找不到视频位置。
