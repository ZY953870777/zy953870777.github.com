markdown语法简介

发表于 2023-04-12 更新于 2023-05-03

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。Markdown支持的语法也不少，博主比较青睐于简单的描述，使用的其实并不多。主要用到的就是： 标题，本地图片， 超链接，列表，代码。

图片

网图

语法：
![alt 属性文本](图片地址 “图片title”)
alt就是显示在图片下面的文字，相当于对图片内容的解释。（注意：Hexo的一些themes是不支持显示标题的，例如正在使用的next主题就不支持。）
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加。
示例：
![这是标题 text](https://img1.baidu.com/it/u=413643897,2296924942&fm=253&fmt=auto&app=138&f=JPEG?w=800&h=500) "这是鼠标悬停时显示的信息")

本地图片

这个方法是hexo提供的，需要在项目配置中打开。如果打开了此配置，那么在新建博客时，除了会生成“博客名.md”的文件外，还会生成一个同名目录，图片需要放在此目录中方可引用。
示例：
{% asset_img test.jpg 这是鼠标悬停时显示的信息 %}

同样的，next主题不支持显示图片标题。

超链接

语法：[超链接名](超链接地址 "超链接title")
示例：[百度传送门](http://baidu.com)
效果：百度传送门

字体

加粗

要加粗的文字左右分别用两个*号包起来
示例： **这是加粗的文字**
效果： 这是加粗的文字

斜体

要倾斜的文字左右分别用一个*号包起来
示例： *这是倾斜的文字*
效果： 这是倾斜的文字

斜体并且加粗

要斜体并且加粗的文字左右分别用三个*号包起来
示例： ***这是斜体加粗的文字***
效果： 这是斜体加粗的文字

删除线

要加删除线的文字左右分别用两个~号包起来
示例： ~~这是加删除线的文字~~
效果： 这是加删除线的文字
